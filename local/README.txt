Το παρόν directory αρχικά είναι κενό σε μια νέα εγκατάσταση της εφαρμογής,
αλλά αμέσως μετά την εγκατάσταση τοποθετούμε σε αυτό το directory files και
directories τα οποία αντιγράφουμε από το directory "lacol". Αμέσως μετά την
αντιγραφή των files και των directories προβαίνουμε σε διορθώσεις των στοιχείων
που περιέχονται στα συγκεκριμένα files προκειμένου τα στοιχεία αυτά να
αντανακλούν το περιβάλλον της νέας εγκατάστασης.
